# 🏛️ 法律荣耀认证生成器

> 一键颁发全球顶级法律荣誉 · 无需业绩 · 无需同行评审 · 只需转账

一个纯娱乐性质的 HTML 单页应用，用于**自嗨式**生成虚构的法律荣誉证书和 3D 奖杯。

## ✨ 功能亮点

- 🎖️ **随机荣誉生成**：每次点击“隆重颁奖”都会随机产生新的奖项名称、颁奖机构、评语和价格。
- 🏆 **3D 动态奖杯**：基于 Three.js 实时渲染的金属质感奖杯，形态随机，可缓慢旋转。
- 📜 **典藏级证书**：仿羊皮纸风格的证书，带有浮雕刻字、防伪印章和可打开的封面。
- 🎉 **庆祝彩蛋**：颁奖时触发飘落彩带和印章动画，仪式感拉满。
- 📋 **一键分享**：自动生成“获奖感言”并复制到剪贴板，方便发朋友圈或社交媒体。

---

## 🚀 如何使用

1. 直接在浏览器中打开 `law_award_generator.html` 即可运行（所有依赖均通过 CDN 加载）。
2. **可选**：在输入框中填写“受奖人姓名”和“律所/机构名称”（不填则使用随机默认值）。
3. 点击 **“隆重颁奖”** 按钮：
   - 3D 奖杯会重新生成新形态。
   - 证书封面优雅翻开，内容同步更新。
   - 彩带飘落，仪式完成。
4. 点击 **“复制获奖感言”** 可将预设文案复制到剪贴板。

> 💡 所有数据均为随机生成，仅供娱乐，不涉及任何真实评选或支付。

## 🛠️ 技术栈

- **HTML5 + CSS3**：响应式布局，仿古书卷风格。
- **JavaScript (ES6)**：业务逻辑与交互控制。
- **Three.js (r128)**：3D 奖杯渲染。
- **Google Fonts**：衬线与无衬线字体搭配，提升质感。
